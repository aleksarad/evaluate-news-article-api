This is a language processing tool that relies on Aylien's Content Analysis API. Please enter a valid URL to an article, blog, or text-based web page below. The content submitted will be evaluated on the following factors:

  • Polarity refers to whether the author's tone is positive, negative, or neutral.
  • Subjectivity analyzes whether the writing is based mostly in the author's opinion, or in fact.
  • The confidence indicator for both polarity and subjectivity measures the statistical confidence that the assessment made by Aylien's language processing is accurate.
